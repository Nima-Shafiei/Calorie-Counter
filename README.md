## 🧠 How It Works

1. Set your daily calorie budget  
2. Add meals or exercises dynamically  
3. Click **Calculate Remaining Calories**  
4. Get instant **Surplus** or **Deficit** feedback  

---

## 🛠️ Tech Stack

- HTML5  
- CSS3 (Flexbox + Gradients + Responsive)  
- Vanilla JavaScript (DOM, Events, Form Handling)
